# 💰 Финансовый трекер

Приложение для учёта личных финансов на ПК и телефоне.

## Возможности

- **Доходы** — добавление доходов по категориям (зарплата, фриланс, инвестиции и др.)
- **Расходы** — учёт расходов по категориям (еда, транспорт, жильё, развлечения и др.)
- **Цели** — создание целей с целевой суммой и отслеживание прогресса накоплений
- **Статистика** — визуализация расходов по категориям за период (неделя/месяц/год)

## Запуск

1. Откройте папку `finance-tracker` в проводнике
2. Запустите локальный сервер (например, через расширение Live Server в VS Code)
3. Или дважды кликните на `index.html` — приложение откроется в браузере

### Рекомендуется: Live Server

Для корректной работы PWA и Service Worker лучше использовать локальный HTTP-сервер:

- **VS Code**: установите расширение "Live Server" и нажмите "Go Live"
- **Node.js**: выполните `npx serve .` в папке проекта
- **Python**: `python -m http.server 8080`

## Установка как приложение

- **Телефон**: откройте в Chrome/Safari → меню → «Добавить на главный экран»
- **ПК**: в Chrome — кнопка установки в адресной строке или меню → «Установить приложение»

## Хранение данных

### Локальный режим (по умолчанию)
Данные хранятся в браузере (localStorage). Работает без интернета.

### Синхронизация ПК + iPhone
Чтобы пользоваться приложением на ПК и телефоне с одними и теми же данными:

1. Создайте проект в [Firebase Console](https://console.firebase.google.com)
2. Включите **Authentication** → Email/Password
3. Создайте **Firestore Database** (режим тестирования)
4. В Firestore → Rules укажите:
   ```
   rules_version = '2';
   service cloud.firestore {
     match /databases/{database}/documents {
       match /users/{userId} {
         allow read, write: if request.auth != null && request.auth.uid == userId;
       }
     }
   }
   ```
5. Добавьте веб-приложение в настройках проекта, скопируйте конфиг в `firebase-config.js`
6. Запустите приложение через HTTPS (например, `npx serve .`)
7. Зарегистрируйтесь на ПК, затем войдите с тем же email на iPhone

После входа данные автоматически синхронизируются между устройствами.
